# OMS
##运营管理系统

###目的:  
用于个人学习，技术整理总结

###项目结构:  
oms  
|--doc 文档、sql  
|--config-repo 注册中心配置文件资源  
|--refistry 注册中心  
|--gateway 网关（spring gateway)  
|--portal 前端业务  
|--core 核心业务  
|--common 公共bean、util  
|--account 单点、认证服务、用户信息相关服务  
|--config 配置中心

###打包方式
jar

###技术选型：  
####前端：  
 jquery  
 bootstarp 3.3  
 thymeleaf
 
####后端：  
spring cloud   
spring boot  
redis  缓存  
Spring Security、OAuth2  权限控制   
Security-OAuth2 密码模式  
spring data jpa 持久化

####数据库：  
mysql

###开发环境

macOS  
intelliJ IDEA  
java 8

###部署环境

red hat linux 6.0
